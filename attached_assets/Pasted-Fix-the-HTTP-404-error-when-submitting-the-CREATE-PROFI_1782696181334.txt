── CAUSE A: API ROUTE MISSING ──────────────

If no route exists for athlete profile creation,
add it to the Express API server file:

  // POST /api/athlete-profiles
  // Create a new athlete profile with invite code
  app.post('/api/athlete-profiles', async (req, res) => {
    try {
      const {
        name,
        dateOfBirth,
        sport,
        club,
        maturityStage,
        programStartDate,
        notes,
        coachId
      } = req.body

      if (!name || !coachId) {
        return res.status(400).json({
          error: 'Name and coachId are required'
        })
      }

      // Generate 6-digit alphanumeric invite code
      const inviteCode = Math.random()
        .toString(36)
        .substring(2, 8)
        .toUpperCase()

      // Set expiry 7 days from now
      const expiresAt = new Date()
      expiresAt.setDate(expiresAt.getDate() + 7)

      // Insert into athlete_profiles table
      const result = await db.insert(athleteProfiles).values({
        coachId,
        name,
        dateOfBirth: dateOfBirth || null,
        sport: sport || 'Tennis',
        club: club || null,
        maturityStage: maturityStage || null,
        programStartDate: programStartDate || null,
        notes: notes || null,
        inviteCode,
        inviteCodeExpiresAt: expiresAt,
        status: 'pending'
      }).returning()

      res.json({
        success: true,
        profile: result[0],
        inviteCode
      })

    } catch (error) {
      console.error('Error creating athlete profile:', error)
      res.status(500).json({ error: error.message })
    }
  })

── CAUSE B: TABLE MISSING IN REPLIT DB ─────

If the athlete_profiles table doesn't exist
in the Replit PostgreSQL database, add it
to the Drizzle schema file (schema.ts or
shared/schema.ts):

  export const athleteProfiles = pgTable(
    'athlete_profiles',
    {
      id: uuid('id').primaryKey().defaultRandom(),
      coachId: integer('coach_id').notNull(),
      createdAt: timestamp('created_at')
        .defaultNow().notNull(),
      name: text('name').notNull(),
      dateOfBirth: date('date_of_birth'),
      sport: text('sport').default('Tennis'),
      club: text('club'),
      maturityStage: text('maturity_stage'),
      programStartDate: date('program_start_date'),
      notes: text('notes'),
      inviteCode: text('invite_code')
        .notNull().unique(),
      inviteCodeExpiresAt: timestamp(
        'invite_code_expires_at'
      ),
      linkedAthleteId: integer('linked_athlete_id'),
      linkedAt: timestamp('linked_at'),
      status: text('status').default('pending')
    }
  )

Then run the migration:
  npm run db:push
