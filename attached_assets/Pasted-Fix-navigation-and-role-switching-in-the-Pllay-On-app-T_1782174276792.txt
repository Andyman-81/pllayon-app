PROBLEM 1 — NO WAY TO SWITCH ROLE OR LOG OUT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Once logged in there is no way to:
  - Log out
  - Switch between roles (athlete / coach / parent)
  - Return to the login/role selection screen

Fix: Add a profile menu to every dashboard.

PROFILE BUTTON
Add a profile/avatar button to the TOP RIGHT of the top nav bar
on all three dashboards (athlete, coach, parent).

Appearance:
  Circle avatar — initials of the logged-in user's name
  Background: current role colour
    Athlete: #10AC6E
    Coach:   #0B7DF1
    Parent:  #D97706
  Size: 36×36px
  Font: Barlow Condensed, white, 14px, uppercase

Tapping the avatar opens a slide-down menu:

  ┌─────────────────────────────┐
  │  [Name]                     │
  │  [Role badge]               │
  │  [Email]                    │
  ├─────────────────────────────┤
  │  Switch Role                │
  │  (shows 3 role pill buttons)│
  │  ATHLETE  COACH  PARENT     │
  ├─────────────────────────────┤
  │  Log Out                    │
  └─────────────────────────────┘

SWITCH ROLE behaviour:
  - Updates localStorage po_role to new selection
  - If user has an account for that role → redirects to that dashboard
  - If user does NOT have an account for that role → redirects to
    that role's registration page (/register/coach etc)
  - Does NOT log out of Supabase auth — just changes the role view

LOG OUT behaviour:
  - Calls supabase.auth.signOut()
  - Clears localStorage (po_role and any cached data)
  - Redirects to /login
  - Role selector is shown fresh on next login
