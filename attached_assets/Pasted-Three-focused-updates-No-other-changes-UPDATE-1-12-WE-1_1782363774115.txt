DATABASE

  create table program_settings (
    id uuid primary key default gen_random_uuid(),
    athlete_id uuid references athletes(id)
      on delete cascade unique,
    program_start_date date,
    updated_at timestamptz default now()
  );

  alter table program_settings enable row level security;
  create policy "settings_own" on program_settings
    using (auth.uid() = athlete_id);

Also allow coach to write (for linked athlete):
  create policy "settings_coach" on program_settings
    for all using (
      exists (
        select 1 from coach_athlete_links
        where coach_id = auth.uid()
        and athlete_id = program_settings.athlete_id
      )
    );

DURATION CALCULATION
When a session is added with time_from and time_to:
  Duration in hours = (time_to - time_from) in minutes / 60
  Round to 1 decimal place
  e.g. 5:00pm to 6:30pm = 1.5h

Store duration as a calculated value when saving:
  Add a duration_hours float column to weekly_schedules:

  alter table weekly_schedules
    add column duration_hours float generated always as (
      case
        when time_from is not null and time_to is not null
        then round(
          (extract(epoch from (
            to_timestamp(time_to, 'HH12:MI AM') -
            to_timestamp(time_from, 'HH12:MI AM')
          )) / 3600)::numeric, 1
        )
        else 0
      end
    ) stored;

If the generated column approach is complex with your
time format, calculate duration_hours in JavaScript
when saving and store it as a regular float column.

UPDATE 3 — INVITE CODE ATHLETE LINKING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Replace the email-lookup linking system with an
invite code system.

Coach creates an athlete profile and generates a
6-digit code. The athlete uses that code to link
their account. No email hunting required.

DATABASE

  create table athlete_profiles (
    id uuid primary key default gen_random_uuid(),
    coach_id uuid references coaches(id)
      on delete cascade,
    created_at timestamptz default now(),

    -- Profile data (coach fills this in)
    name text not null,
    date_of_birth date,
    sport text default 'Tennis',
    club text,
    maturity_stage text,
      -- 'pre-phv', 'circa-phv', 'post-phv'
    program_start_date date,
    notes text,

    -- Linking
    invite_code text unique not null,
    invite_code_expires_at timestamptz,
    linked_athlete_id uuid references athletes(id),
    linked_at timestamptz,
    status text default 'pending'
      -- 'pending', 'linked', 'expired'
  );

  alter table athlete_profiles enable row level security;

  -- Coach can manage their own profiles
  create policy "profiles_coach_own" on athlete_profiles
    using (auth.uid() = coach_id);

  -- Athlete can read profile with matching invite code
  -- (used during linking flow)
  create policy "profiles_athlete_link" on athlete_profiles
    for select using (true);
    -- Public read for code lookup — coach_id and sensitive
    -- data protected by application logic

ATHLETE FLOW — ENTERING THE CODE

On the athlete registration form (/register/athlete),
after the main profile fields are complete,
add a new section below the form:

  Divider: "HAVE A COACH INVITE CODE?"

  Label: "ENTER YOUR CODE (OPTIONAL)"
  hint: "Your coach will have given you a 6-digit code.
         You can also enter it later from your dashboard."

  Input:
    Monospace font (Space Mono)
    Letter-spacing wide
    Text transforms to uppercase automatically
    Max length: 6 characters
    placeholder: "e.g. PL4829"

  On registration complete:
    If code was entered:
      Look up athlete_profiles by invite_code
      Check code is not expired and status = 'pending'
      If valid:
        Update athlete_profiles:
          linked_athlete_id = new athlete's id
          status = 'linked'
          linked_at = now()
        Insert into coach_athlete_links:
          coach_id = athlete_profiles.coach_id
          athlete_id = new athlete's id
        Show success: "Linked to [Coach Name] ✓"
      If invalid or expired:
        Show: "Code not found or expired.
               Ask your coach for a new one."
        Continue registration without linking

  If no code entered:
    Register normally
    On athlete dashboard show a persistent
    (but dismissible) banner:
      "Have a coach invite code? Enter it here →"
      → /athlete/enter-code
